# Data Models Specification (Go + TypeScript)

Data flows between the Go backend and React frontend as JSON (auto-serialized by Wails). Go structs use `json` tags for field mapping. TypeScript types are auto-generated by Wails CLI from Go struct definitions.

## 1. Connection Models

### `DatabaseConnection` (Go)
```go
type DatabaseConnection struct {
    ID        uuid.UUID    `json:"id"`
    Name      string       `json:"name"`
    Type      DatabaseType `json:"type"`       // "postgres", "mysql", etc.
    Group     string       `json:"group"`
    ColorTag  string       `json:"colorTag"`   // "red", "blue", "green"

    // Core
    Host          string `json:"host"`
    Port          int    `json:"port"`
    Database      string `json:"database"`
    Username      string `json:"username"`
    LocalFilePath string `json:"localFilePath"` // SQLite/DuckDB

    // SSH
    SSH SSHTunnelConfig `json:"ssh"`
    // SSL
    SSL SSLConfig `json:"ssl"`

    // Advanced
    SafeMode       SafeModeLevel `json:"safeMode"`
    StartupCommand string        `json:"startupCommand"`
    PreConnectScript string      `json:"preConnectScript"`
}
// Password is NEVER in this struct — stored in OS Keychain via go-keyring
```

### `ConnectionSession` (Go — runtime only, not serialized)
```go
type ConnectionSession struct {
    ConnectionID uuid.UUID
    Status       ConnectionStatus // disconnected, connecting, connected, error
    ActiveDB     string
    Driver       DatabaseDriver
    SSHTunnel    *SSHTunnel
    LastPingAt   time.Time
}
```

## 2. Editor & Tab Models

### `QueryTab` (Go + TypeScript)
```go
type QueryTab struct {
    ID       uuid.UUID `json:"id"`
    Type     TabType   `json:"type"`     // "table", "query", "structure"
    Title    string    `json:"title"`
    Query    string    `json:"query"`

    // Table-specific
    SchemaName string `json:"schemaName"`
    TableName  string `json:"tableName"`
    IsView     bool   `json:"isView"`
    DatabaseName string `json:"databaseName"`

    // Results (populated after query execution)
    Columns       []ColumnInfo `json:"columns"`
    Rows          [][]any      `json:"rows"`       // 2D array for grid
    TotalRowCount int64        `json:"totalRowCount"`
    Offset        int          `json:"offset"`
    ExecutionTime float64      `json:"executionTime"`
    ErrorMessage  string       `json:"errorMessage"`
    IsExecuting   bool         `json:"isExecuting"`
}
```

### `PersistedTab` (disk-only, no result data)
```go
type PersistedTab struct {
    ID           uuid.UUID `json:"id"`
    Title        string    `json:"title"`
    Query        string    `json:"query"`
    Type         TabType   `json:"type"`
    TableName    string    `json:"tableName"`
    IsView       bool      `json:"isView"`
    DatabaseName string    `json:"databaseName"`
}
```

## 3. Schema Metadata Models

```go
type DatabaseInfo struct {
    Name      string `json:"name"`
    SizeBytes int64  `json:"sizeBytes"`
}

type SchemaInfo struct {
    Name     string      `json:"name"`
    Tables   []TableInfo `json:"tables"`
    Views    []TableInfo `json:"views"`
    Routines []RoutineInfo `json:"routines"`
}

type ColumnMetadata struct {
    Name            string `json:"name"`
    Type            string `json:"type"`
    IsNullable      bool   `json:"isNullable"`
    IsAutoIncrement bool   `json:"isAutoIncrement"`
    IsPrimaryKey    bool   `json:"isPrimaryKey"`
    DefaultValue    any    `json:"defaultValue"`
    Comment         string `json:"comment"`
}
```

## 4. Settings & Preferences

```go
type AppSettings struct {
    Theme           string  `json:"theme"`    // "system", "light", "dark"
    FontSize        int     `json:"fontSize"`
    FontFamily      string  `json:"fontFamily"`
    WrapLines       bool    `json:"wrapLines"`
    ShowLineNumbers bool    `json:"showLineNumbers"`
    VimMode         bool    `json:"vimMode"`
    AutoCapitalize  bool    `json:"autoCapitalize"`
    Autocomplete    bool    `json:"autocomplete"`
    QueryTimeout    int     `json:"queryTimeout"` // seconds
    RowsPerPage     int     `json:"rowsPerPage"`  // default 500
}

type AISettings struct {
    Provider      string  `json:"provider"` // "openai", "anthropic", "ollama"
    APIKey        string  `json:"-"`        // Never serialized — stored in Keychain
    CustomBaseURL string  `json:"customBaseUrl"`
    Model         string  `json:"model"`
    Temperature   float64 `json:"temperature"`
}
```

## 5. TypeScript Types (Auto-generated by Wails)
Wails CLI generates TypeScript bindings from Go structs in `frontend/src/wailsjs/go/models.ts`. These are used directly by React components — ensuring type-safety across the full stack.
